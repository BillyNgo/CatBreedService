Description
----------------------------------------------------------------------------------------------------------------------
Sample .NET Core Web API application implemented with CQRS, Mediator Pattern approach and Domain Driven Design.

## Techical Stack:
- ASP.NET Core 6.0 (with .NET Core 6.0)
- ASP.NET Core WebApi 
- Entity Framework Core
- .NET Core Native DI
- AutoMapper
- FluentValidator
- MediatR
- Swagger UI
- Azure Cosmos Db Emulator
- xUnit
- Moq
- Fluent Assertions
- Logging
